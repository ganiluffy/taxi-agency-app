# Taxi Agency Booking Application íº–

A full-stack taxi booking and fare estimation platform built using:

- React (Frontend)
- Node.js + Express (Backend)
- FastAPI (Python Microservice)
- MongoDB (Database)

## Features
- Fare estimation using maps
- Taxi booking (ride / full-day)
- JWT & Google authentication
- Admin dashboard
- Payment tracking

Built as a real-world project for a taxi agency.
